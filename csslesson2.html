What is a div element?

A bucket-like element that groups sections of a webpage together. It stands for division and divides the page into sections.

What is one way you can identify or "target" a div element?

You can identify or target a div by using its class name in CSS. You write a period followed by the class name.

Why would you use a div element?

You would use a div element to group related content together so you can style or organize it easily. By giving the div a class and targeting it in CSS, you can apply the same design or layout to multiple sections on a webpage.

What is a border property?

The border property in CSS is used to add and style a border around an element. It controls the border’s width, style, and color, helping define the element’s size and shape visually.

What are some different border properties?

  1. Border Width
  2. Border Color
  3. Border Sides
  4. Rounded Borders

What is the difference between margins and padding?

Margin is the space outside the element’s border, pushing the element away from other elements. Padding is space inside the element’s border, pushing the content away from the border.
